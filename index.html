IDE API

that website is have advanced application programming interface (API) you can use the API to customize the website using the url link and the search params and it the existing params inside the url link like that link:-

https://ide-editor.github.io?mode=application/json

use the mode param for tell the editor any mode we need to use it you write the mode as a MIME type of the language for example i want to use the JavaScript language in the editor so i will write in the url:-

https://ide-editor.github.io?mode=application/json&lineNumbers=true

use the lineNumbers param for tell the editor if we want to see the linenumbers or no't

https://ide-editor.github.io?mode=application/json&
lineNumbers=true&matchBrackets=true

use the matchBrackets param for tell the editor if we want to active the matchBrackets or no't

https://ide-editor.github.io?mode=application/json&lineNumbers=true
&matchBrackets=true&readOnly=true

use the readOnly param for make the text dosen't changing into the editor and this attribute for protect the codes

https://ide-editor.github.io?mode=application/json&lineNumbers=true
&matchBrackets=true&readOnly=true&theme=IDE

use the theme param for set theme of the editor

https://ide-editor.github.io?mode=application/json&lineNumbers=true
&matchBrackets=true&readOnly=true&theme=IDE
&SourceCode=Base64 String

use the SourceCode param for set the existing source code into the editor using Base64 String
and for do this you have to convert the text to URL Encoding using the encodeURI(String) and then convert this result to Base64 String and for do this you can use btoa(String)
